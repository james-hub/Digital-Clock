Information:
Digital Clock is a simple php script.
It displays a a clock with the current date/time and it updates itself every second.
You can choose which format you will show to your visitors.
There are many examples in the 'time examples.txt' file.

Setup:
The setup is very simple,
Upload the script and the map to your server.
Edit the file 'time_view.php' in the format you want. (there are many examples in the 'time examples.txt' file)
include the file 'time_script.php'
=====================================
<?php
include 'time_script.php';
?>
=====================================
And you are done.
